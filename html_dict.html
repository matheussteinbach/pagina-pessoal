Página Inicial
Tag !DOCTYPE
Usada para definir o tipo do documento e a versão sendo utilizada.

Tag html
Define o início e fim do documento HTML, envolve todo o documento e define o idioma do conteúdo.

Tag head
Usada para fornecer informações sobre o documento que não são diretamente exibidas na tela do navegador, além de conter metadados como titulo, links para o CSS e JS, imagens, entre outros.

Tag meta
Usada para fornecer metadados sobre o documento como charset, palavras-chave, entre outros.

Tag title
Define o título da página que aparece na guia do navegador.

Tag link
Vincular recursos externos ao documento HTML.

Tag body
Definir o conteúdo principal da página e todo o conteúdo visível.

Tag header
Definir cabeçalho da página, seção no começo.

Tag nav
Definir uma seção de navegação para a página.

Tag menu
Menu de comandos ou opções de navegação.

Tag li
Definir um item da lista dentro das tags, ul, ol ou menu.

Tag a
Criar links (âncoras) para outras páginas da web, arquivos, seções da mesma página ou recursos externos.

Tag aside
Seção para adicionar um conteúdo secundário, tangencial, porém relacionado ao conteúdo da página.

Tag h1
Título (Heading/Cabeçalho) de uma seção da página, título principal.

Tag img
Inserir imagens em uma página web. Tag vazia com atributo obrigatório "src".

Tag p
Definir parágrafos em um texto.

Tag div
Criar uma divisão ou contêiner genérico.

Tag main
Usado para marcar e envolver o conteúdo principal da página web.

Tag section
Usado para definir seção ou área temática.

Tag br
Inserir uma quebra de linha.

Tag h2
Subtítulos ou títulos de seções secundárias da página.

Tag ul
Define uma lista não ordenada.

Tag strong
Definir um texto importante, exibindo em negrito.

Tag footer
Definir rodapé da página, seção no final.

Tag dl
Lista de definição, envolve toda a lista e usada para definir contexto

Tag dt
Definir um termo na lista de definição (chave)

Tag dd
Definir uma definição (valor) na lista de definição
